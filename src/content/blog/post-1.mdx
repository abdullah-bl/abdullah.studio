# My First MDX Post

Welcome to my first blog post! This is written in MDX format, which allows me to combine Markdown with React components.

## What is MDX?

MDX is a powerful format that lets you write JSX in your Markdown documents. This means you can:

- Write content in Markdown
- Import and use React components
- Create interactive content
- Maintain a great writing experience

## Getting Started

To create a new blog post, simply add a new `.mdx` file to the `src/blog/` directory with the proper frontmatter:

```mdx
---
title: "Your Post Title"
description: "A brief description of your post"
pubDate: 2024-01-15
---

# Your Content Here

Your markdown content goes here...
```

## Features

This blog supports:
- **Markdown syntax** - All standard Markdown features
- **React components** - Import and use custom components
- **Frontmatter** - Metadata for each post
- **Automatic routing** - Posts are automatically available at `/blog/[slug]`
